STEP 1: LINUX + BASICS (COMPLETE WITHOUT PREMIUM)

Since Linux Fundamentals 2 and 3 are locked, do this instead:

1. Practice in Kali Linux daily:
- Navigate directories (ls, cd, pwd)
- Read files (cat, less)
- Edit files (nano)
- Permissions (chmod, chown)
- Processes (ps, top)
- Networking basics (ip a, netstat)

2. Search on YouTube:
- “Linux command line for beginners full course”
- “Linux permissions explained”
- “Linux networking basics”

Goal: Be comfortable using terminal without confusion
